# Calculadora React

Aplicación de calculadora construida con **React 18** y **Bootstrap 5**.  
Diseño dark con tipografía monoespaciada y animaciones CSS.

## Instalación

```bash
npm install
npm start
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.
